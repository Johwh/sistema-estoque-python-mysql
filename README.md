
# Sistema de Estoque com Python + MySQL

Projeto desenvolvido para praticar integração entre Python e MySQL.

## Tecnologias

- Python
- MySQL
- mysql-connector-python
- SQL
  
## Funcionalidades

- Conexão com banco de dados
- Cadastro de produtos
- Consulta de produtos
- Estrutura CRUD

## Como executar

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python main.py
```
